# SerialFlow Studio

SerialFlow Studio is a Windows desktop tool for USB-to-serial communication, terminal monitoring, command sending, and serial log recording.

## Downloads

- `SerialFlow_Studio.exe` - Windows application
- `SerialFlow_Studio_User_Guide.pdf` - setup and usage guide
- `SHA256SUMS.txt` - file checksums for verification

## Quick Start

1. Download `SerialFlow_Studio.exe`.
2. Connect your USB-to-serial adapter or target board.
3. Open the application.
4. Select the COM port and baud rate.
5. Click **Connect**.

For the full workflow, see `SerialFlow_Studio_User_Guide.pdf`.

## System Requirements

- Windows 10 or later
- USB-to-serial adapter driver installed
- Available COM port
